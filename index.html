This page was made with HTML and CSS, using a grid layout to organize the page. I used the following resources to help me make this page: W3Schools CSS grid, W3Schools HTML and Css, and Professor Dulimarta's lecture notes. The grid is 4x4, sized explicitly and with grid elements that span multiple rows and columns.

I made use of HSL, rgba, and hex color codes to make the page more visually appealing. Each grid element has a gradient applied to it as well. The icons at the bottom are svg.

The page is using a custom google font, Source Code Pro.
